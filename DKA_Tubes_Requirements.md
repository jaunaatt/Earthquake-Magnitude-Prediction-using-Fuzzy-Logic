# DKA TUBES: FUZZY LOGIC IMPLEMENTATION

## Group Requirements
* Assignments must be completed in groups (maximum 3 people)
* Each group must have:
  * A different problem topic
  * A different dataset
* Topics and datasets must be approved by the lecturer
* Duplication between groups in all classes is prohibited.

## Dataset Requirements
* Datasets must be real data
* The use of synthetic/artificial datasets is prohibited
* The dataset must consist of at least 5,000 rows of data, each row containing at least 5 input variables and 1 output variable.
* Must include a link to the dataset source.

## Fuzzy Logic Requirements
* **Design:**
  * Linguistic variables
  * Membership functions
  * Rule base (minimum 15)
* **Implement:**
  * Mamdani Fuzzy
  * Sugeno Fuzzy
* **Perform the following processes:**
  * Fuzzification
  * Inference
  * Defuzzification
* Using fuzzy libraries is not permitted; it must be from scratch.
* **Analysis that MUST be performed:**
  * Comparison: Mamdani vs. Sugeno
  * Evaluation:
    * Differences in output results
    * Accuracy or other performance metrics
    * Interpretation (advantages and disadvantages of each method)

## Additional notes regarding performance:
* System performance evaluation is tailored to the type of problem and the data used.
* If the dataset has labels (ground truth), students can use metrics such as accuracy or other classification metrics.
* If the output is a continuous value (e.g., price predictions or scores), use error metrics such as MAE (Mean Absolute Error) or MSE (Mean Squared Error).
* If ground truth is not available, the evaluation is conducted qualitatively, assessing whether the output is reasonable and consistent with the established rules.
